# Changelog

## [v1.5.0] - 2026-04-07
### ✨ Features
- **LCD 2004 Support:** Добавлена поддержка 4-строчного I2C-дисплея (20x4).
- **Versioning:** Добавлен дефайн версии `VERSION` для удобного отслеживания обновлений.
- **UI:** На экране теперь отображается название проекта и текущая версия при старте.

### 🛠 Stability Fixes
- **Baud Rate:** Снижена до **9600** для максимальной надежности обмена данными (как в nano1602).
- **Sync Fix:** Убрано `while(!Serial)`, приводившее к зависанию ESP8266 при запуске из Linux-консоли.
- **Raw Mode:** Улучшен алгоритм чтения данных из порта для корректной обработки потока.

## [v1.3.0] - 2026-04-03
### ✨ Features
- **start.sh:** Добавлен скрипт для автоматической проверки и установки Python3/pip.
- **Auto-detect:** `monitor.py` теперь сам находит USB-порт ESP-12E.
- **Baud Rate:** Добавлена поддержка выбора скорости через аргументы командной строки.

## [v1.2.0] - 2026-04-03
### 📝 Documentation
- **Install Guide:** Добавлена инструкция по скачиванию проекта (Git и ZIP) в README.

## [v1.1.0] - 2026-04-03
### 💻 PC Monitor
- **monitor.py:** Добавлен Python-скрипт для красивого вывода температуры и влажности в консоль Linux.
- **Emoji Output:** Данные выводятся с иконками для лучшей читаемости.

## [v1.0.0] - 2026-04-03
### 🚀 Initial Release
- **ESP-12E + AHT10:** Базовая прошивка для считывания температуры и влажности.
- **Serial Output:** Вывод данных в консоль (115200 baud).
- **Docs:** Первый вариант README с схемой подключения.
